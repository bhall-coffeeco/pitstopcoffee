PIT STOP COFFEE WEBSITE V2

UPLOAD
1. Unzip this folder.
2. Open the GitHub repository that currently hosts pitstopcoffee.co.
3. Upload every file and the entire assets folder to the repository root.
4. Choose "Commit changes."
5. GitHub Pages normally republishes the website within a few minutes.

IMPORTANT
- The inquiry form sends to: info@pitstopcoffee.co
- The form opens the visitor's email app. It does not store customer information.
- The CNAME file must stay in the repository so pitstopcoffee.co remains connected.

PRICING
The site currently explains how pricing is calculated and promises a written quote without a phone call.
It does not invent package prices because no event rates were provided.
When launch rates are finalized:
- Update the "Pricing without the mystery" section in index.html.
- Replace the line: "Starting package prices will be posted here..."
- A future instant estimator can use guest count, service time, travel, and service style.

PHOTOS
- Current logo: assets/pit-stop-logo.webp
- Current menu/cart image: assets/menu-display.webp
- To replace the menu image later, export a new photo using the same filename.
- Actual farmers-market and drink photos can later be added as a gallery without changing the core site.

COMMON TEXT EDITS
- Menu drinks: search index.html for "Signature menu"
- Indoor FAQ: search index.html for "Can the cart operate indoors?"
- Story text: search index.html for "Our story"
- Email: search the entire project for "info@pitstopcoffee.co"

FILES
- index.html: content and search metadata
- styles.css: design and mobile layout
- script.js: mobile menu, FAQ, animations, and inquiry email
- assets/: optimized images
- CNAME: custom domain connection
- robots.txt and sitemap.xml: search engine files
